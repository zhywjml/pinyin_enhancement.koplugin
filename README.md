# pinyin_enhancement.koplugin

#### 介绍
koreader插件-拼音输入法增强：在拼音输入法上方显示候选词栏，点击即可输入汉字

#### 安装教程

下载解压后将pinyin_enhancement.koplugin文件夹放到koreader\plugins目录下

#### 使用说明

1.  [设置]-[拼音输入法增强]-选择启用拼音候选词（默认启用状态，可选择关闭）
2.  候选栏左侧拼音按键会实时显示用户输入的拼音，点击该按键可一键清空当前拼音状态
3.  候选词来自koreader源码表ui/data/keyboardlayouts/zh_pinyin_data，理论上可以通过置换源码表（即添加首字母映射）实现首字母拼音模式
